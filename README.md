# Casa do Norte Big Jabá - Site Oficial

## 📋 Descrição do Projeto
Site oficial do restaurante **Casa do Norte Big Jabá**, desenvolvido como parte de um projeto extensionista de 100 horas da PUCPR. O site apresenta um design moderno e responsivo, com todas as informações necessárias sobre o restaurante especializado em comida nordestina.

## 🎯 Objetivo
Criar uma presença digital simples e funcional para o restaurante Casa do Norte Big Jabá, permitindo que os clientes:
- Conheçam o cardápio completo com preços
- Visualizem os horários de funcionamento
- Entrem em contato via WhatsApp
- Encontrem a localização do restaurante
- Conheçam a história do projeto extensionista

## 🏪 Sobre o Restaurante
- **Nome:** Casa do Norte Big Jabá
- **Localização:** Jardim Três Marias, São Paulo/SP
- **Especialidade:** Comida Nordestina
- **Equipe:** 5 pessoas
- **Horário:** Segunda a Domingo - 11h às 22h
